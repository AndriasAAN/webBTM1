# **App Name**: Batumarta 1 Digital Hub

## Core Features:

- Admin Authentication: Secure admin login using Firebase Authentication, restricting access to the specified email.
- News Management: CRUD interface for managing news articles: create, read, update, delete with rich text editor and thumbnail upload to Storage.
- Photo Gallery: Upload and manage photos in a gallery, with options to set photos for the home page slider.
- Front-End Customization: Admin panel to customize website appearance, including header image, tagline, and theme colors, saved in Firestore.
- Homepage Carousel: Displays images in a carousel/slider on the homepage.
- News Display: Displays 3 of the newest articles.
- Firestore Rules: Firestore and Storage rules configured so only admins can make changes, while all users can view news and the photo gallery.

## Style Guidelines:

- Primary color: Soft pink (#FF7EBD) to evoke a sense of community and warmth.
- Background color: Very light pink (#FFE0F0), almost white, for a clean and airy feel.
- Accent color: Rose pink (#E91E63), for buttons and highlights, providing a modern touch.
- Font: 'Poppins', a geometric sans-serif font, should be used throughout the website, creating a contemporary feel. Note: currently only Google Fonts are supported.
- Clean and modern icons representing different aspects of the village, maintaining consistency with the pink color scheme.
- A responsive and clean layout, ensuring the website is accessible on all devices.
- Subtle transitions and animations to enhance user experience without overwhelming the user.
